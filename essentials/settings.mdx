# Global Settings

Configure global settings for your Gx402 SDK project to optimize performance, security, and functionality across all platforms.

## Configuration Overview

The Gx402 SDK is configured through platform-specific initialization methods, but all share common configuration principles.

### Basic Configuration Structure

#### WebGL/JavaScript
```javascript
const config = {
  // Required settings
  apiKey: 'your-api-key',
  
  // Environment settings
  environment: 'production', // 'development' or 'production'
  
  // Feature flags
  enableRealtimeSync: true,
  debug: false,
  enableEncryption: true,
  
  // Performance settings
  syncInterval: 1000, // milliseconds
  maxRetries: 3,
  
  // Security settings
  enableCertificatePinning: true
};

const Gx402 = new Gx402SDK(config);
await Gx402.init();
```

#### Unity (C#)
```csharp
var config = new Gx402Config {
    ApiKey = "your-api-key",
    Environment = Gx402Environment.Production,
    EnableRealtimeSync = true,
    LogLevel = Gx402LogLevel.Info,
    SyncInterval = 1000
};

Gx402Manager.Initialize(config);
```

#### Unreal Engine (C++)
```cpp
FGx402Config Config;
Config.ApiKey = TEXT("your-api-key");
Config.Environment = EGx402Environment::Production;
Config.bEnableRealtimeSync = true;
Config.SyncInterval = 1000;

UGx402Manager::Initialize(Config);
```

## Environment Configuration

### Development vs Production

#### Environment-Specific Settings
```javascript
const getEnvironmentConfig = (env) => {
  const configs = {
    development: {
      apiKey: process.env.Gx402_DEV_KEY,
      environment: 'development',
      debug: true,
      enableRealtimeSync: false, // disable during development
      logLevel: 'verbose',
      apiEndpoint: 'https://dev-api.Gx402.com/v1'
    },
    staging: {
      apiKey: process.env.Gx402_STAGING_KEY,
      environment: 'development', // staging uses dev environment
      debug: true,
      enableRealtimeSync: true,
      logLevel: 'info'
    },
    production: {
      apiKey: process.env.Gx402_PROD_KEY,
      environment: 'production',
      debug: false,
      enableRealtimeSync: true,
      logLevel: 'warn'
    }
  };
  
  return configs[env] || configs.development;
};
```

## Sync Configuration

### Sync Strategies

#### Optimistic vs Pessimistic Sync
```javascript
// Optimistic sync (default): Apply locally immediately, sync in background
Gx402.configure({
  syncStrategy: 'optimistic',
  enableImmediateUiUpdate: true
});

// Pessimistic sync: Wait for server confirmation
await Gx402.sync('data', payload, { 
  strategy: 'pessimistic',
  waitForConfirmation: true 
});
```

### Conditional Configuration
```javascript
const getDynamicConfig = () => {
  const isMobile = /Android|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(
    navigator.userAgent
  );
  
  const isLowEndDevice = navigator.hardwareConcurrency < 4;
  
  return {
    syncInterval: isLowEndDevice ? 2000 : 1000,
    enableCompression: true,
    maxConcurrentOperations: isLowEndDevice ? 5 : 10
  };
};

await Gx402.configure(getDynamicConfig());
```

## Configuration Validation

### Validate Configuration
```javascript
// Validate configuration before applying
const validateConfig = (config) => {
  const errors = [];
  
  if (!config.apiKey) {
    errors.push('API key is required');
  }
  
  if (config.syncInterval < 100) {
    errors.push('Sync interval must be at least 100ms');
  }
  
  if (!['development', 'production'].includes(config.environment)) {
    errors.push('Environment must be development or production');
  }
  
  return {
    valid: errors.length === 0,
    errors
  };
};

const validation = validateConfig(config);
if (!validation.valid) {
  console.error('Configuration validation failed:', validation.errors);
}
```

## Configuration Management

### Environment-Based Configuration Management
```javascript
class ConfigManager {
  constructor() {
    this.config = this.loadConfig();
  }
  
  loadConfig() {
    // Load from environment variables, local storage, or defaults
    return {
      apiKey: process.env.Gx402_API_KEY || this.getStoredApiKey(),
      environment: process.env.NODE_ENV || 'development',
      enableRealtimeSync: this.getStoredSetting('syncEnabled', true),
      debug: process.env.NODE_ENV !== 'production'
    };
  }
  
  async applyToGx402(Gx402) {
    await Gx402.configure(this.config);
  }
  
  getStoredApiKey() {
    return localStorage.getItem('Gx402_api_key');
  }
  
  getStoredSetting(key, defaultValue) {
    const stored = localStorage.getItem(`Gx402_${key}`);
    return stored !== null ? JSON.parse(stored) : defaultValue;
  }
}

const configManager = new ConfigManager();
await configManager.applyToGx402(Gx402);
```
