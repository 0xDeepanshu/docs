## Getting Started by Platform

Pick your development environment and follow the setup steps below.

<Steps>
<Step title="1. Select Your Platform">
<Tabs>
  <TabItem value="unity" label="Unity">
    Use the Gx402 Unity package to access the full SDK inside your game engine. Supports C# and async API calls.
  </TabItem>

  <TabItem value="unreal" label="Unreal Engine">
    Integrate Gx402 in your Unreal Engine project using our Blueprint and C++ SDKs.
  </TabItem>

  <TabItem value="webgl" label="WebGL">
    Bring x402 features to browser-based games with our lightweight JavaScript library.
  </TabItem>

  <TabItem value="mobile" label="Mobile">
    Native SDKs for both **iOS** and **Android**, optimized for performance and secure mobile payments.
  </TabItem>
</Tabs>
</Step>

<Step title="2. Install the SDK">
  Follow our platform-specific installation guides to set up the Gx402 SDK. Each guide includes code snippets, dependencies, and testing examples.
</Step>

<Step title="3. Integrate and Build">
  Connect your game logic to Gx402 APIs — from payments and identity to player data and assets.  
  Once integrated, you can deploy instantly across all supported platforms.
</Step>
</Steps>

## What You Can Build

With Gx402, you can:

- Enable **in-game purchases** and **microtransactions** directly through x402.
- Sync **player data** between devices using real-time APIs.
- Add **authentication and user linking** across your ecosystem.
- Integrate with **Telegram**, **Farcaster**, or **Base apps** for cross-platform play.
- Build **Web3-connected games** without the headache of managing multiple SDKs.
